Feathercoin-php
===========

A [Feathercoin][Feathercoin] library for [PHP](http://www.php.net/).

Requirements
------------

### PHP requirements:
* PHP5
* cURL support  
* SSL support (if you're using HTTPS to talk to feathercoind)
